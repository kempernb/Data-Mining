Preparation

Splitting Data into Training and Validation Sets

This is one of the principles of data mining and machine learning.

We’ll first drop the record containing the outlier for CC that we discussed during lecture, as well as the columns for ID (not a useful predictor), Model (too many levels), MFG_Year (very highly correlated with Age_08_04), and Cylilnders (value is always 4).

toyota <- toyota[toyota$CC != 16000, -c(1:2, 6, 15)]

We will sample 80% of the original data and use it as the training set and use the remaining 20% as the validation set. The regression model will be built using the training set, and its future performance will be evaluated using the validation set. (Here, we try a slightly different partitioning method from what we did in lecture. Feel free to use whatever method you prefer.)

set.seed(1)
train.rows <- sample(nrow(toyota), nrow(toyota) * 0.8)
toyota.train <- toyota[train.rows, ]
toyota.valid <- toyota[-train.rows, ]

(Optional) Standardization

If we want our results to be invariant to the units and the parameter estimates \(\beta_i\) to be comparable, we can standardize the variables. Essentially, we are replacing the original values with their z-scores.

One method: Create new variables manually

toyota$sd.km <- (toyota$KM - mean(toyota$KM)) / sd(toyota$KM)

Another method: Use the scale() function

toyota$sd.km <- scale(toyota$KM)

Usually, if necessary, you may want to standardize all predictors. (If you run this code, your regression coefficients will look different from what you see below. It is not necessary to standardize in order to perform this analysis.)

toyota[, -c(1,5,8)] <- scale(toyota[, -c(1,5,8)])

This technique is not as important in linear regression because it will only affect the interpretation but not the model estimation and inference.

Prediction

To predict Price given a new set of records, we use predict().

pred.model0 <- predict(model0, newdata = toyota.valid)

Be careful that the argument “newdata” must be a data frame. You can always convert a table, vector, or matrix to a data frame using data.frame().
